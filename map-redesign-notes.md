# Map Redesign Analysis

## Reference Image Analysis

The user wants a political-style map of South America with the following characteristics:

### Map Style
- Political map (not satellite) with countries in green tones and ocean in light blue
- Country names visible (Brasil, Peru, Chile, Argentina, Bolívia, Paraguai, Uruguai, etc.)
- Brazilian state abbreviations visible (SP, PR, SC, RJ, MG, ES, PE, PB, etc.)
- State/country borders shown with dashed or solid lines
- Clean, modern cartographic style

### Marker Locations (Purple dots)
From the reference image, the purple dots are located at:
1. **Pernambuco (PE)** - dot near the coast of PE state (Recife area)
2. **Paraíba (PB)** - dot near PB coast
3. **São Paulo (SP)** - dot in SP state area
4. **Rio de Janeiro (RJ)** - dot in RJ area
5. **Espírito Santo (ES/MG area)** - dot between MG and ES
6. **Paraná (PR)** - dot in PR state
7. **Santa Catarina (SC)** - dot in SC state
8. **Peru (Lima)** - dot on Peru's coast
9. **Paraguai** - dot near Paraguai/PR border area
10. **Argentina (Buenos Aires)** - dot in southern Argentina area
11. **Uruguai (Buenos Aires area)** - dot near Buenos Aires/Uruguai

### Dot Style
- Solid purple/dark blue circles
- No labels next to the dots (the map itself has the labels)
- No glow or ping animation visible
- Simple, clean dots

### Key Differences from Current Implementation
1. Map type: Political map instead of satellite image
2. More locations marked (includes RJ, ES/MG, PB, Paraguai)
3. Dots are plain purple without labels (map provides context)
4. No animation effects on dots
